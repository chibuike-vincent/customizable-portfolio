# Images Directory

This directory should contain placeholder/dummy images that will be used as defaults.

## Required Images

Place these images in this directory:

1. **dummy-profile.jpg** (Recommended: 800x800px)
   - Used for profile images when none is uploaded
   - Square format recommended

2. **dummy-achievement.jpg** (Recommended: 800x600px)
   - Used for achievement images when none is uploaded
   - Landscape format recommended

3. **dummy-philanthropic.jpg** (Recommended: 800x600px)
   - Used for philanthropic activity images when none is uploaded
   - Landscape format recommended

4. **dummy-message.jpg** (Recommended: 400x400px)
   - Used for goodwill message user images when none is uploaded
   - Square format recommended

## Image Sources

You can get placeholder images from:
- Unsplash (https://unsplash.com)
- Pexels (https://www.pexels.com)
- Placeholder.com
- Or create your own

## Notes

- Images uploaded through the admin panel will be stored in `public/uploads/`
- Default images should be professional and appropriate for a political website
- Consider image optimization for better performance
